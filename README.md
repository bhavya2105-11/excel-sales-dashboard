# excel-sales-dashboard
# Project Description

* Built an interactive **Sales Dashboard** in **Microsoft Excel** to analyze retail sales data.
* Cleaned and organized raw data to improve accuracy and consistency.
* Used **Pivot Tables**, **Pivot Charts**, and **Slicers** to create dynamic and interactive reports.
* Visualized key metrics such as sales, orders, customer demographics, and monthly trends.
* Analyzed **state-wise**, **category-wise**, and **channel-wise** sales performance.
* Applied **Conditional Formatting** and dashboard design principles for better readability.
* Enabled users to filter data and explore insights through an intuitive interface.
* Demonstrated skills in **Microsoft Excel**, **Data Cleaning**, **Data Analysis**, and **Data Visualization**.
